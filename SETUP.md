# Инструкция по настройке бота

## 📋 Изменения в .env файле

Добавьте следующие переменные в файл `.env`:

```env
BOT_TOKEN=ваш_токен_бота
CHANNEL_ID=@ваш_канал
GAME_TIMER_DURATION=150
```

### Описание переменных:

- **BOT_TOKEN** - Токен вашего Telegram бота (уже должен быть)
- **CHANNEL_ID** - ID канала для проверки подписки. Может быть:
  - `@channelname` - для публичных каналов
  - `-1001234567890` - для приватных каналов (числовой ID)
  - Оставьте пустым `""`, если проверка подписки не нужна
- **GAME_TIMER_DURATION** - Длительность игры в секундах (по умолчанию 150 = 2.5 минуты)

## 🚀 Команды для запуска

### 1. Установка зависимостей (если еще не установлены)

```bash
poetry install
```

### 2. Применение миграций базы данных

```bash
poetry run alembic upgrade head
```

### 3. Запуск бота

```bash
poetry run python main.py
```

или

```bash
python main.py
```

## ⚙️ Настройка канала

1. Создайте канал в Telegram (или используйте существующий)
2. Добавьте бота в канал как администратора
3. Укажите ID канала в `.env`:
   - Для публичного канала: `@channelname`
   - Для приватного: получите числовой ID через бота @userinfobot

## 🎮 Использование

После запуска бота:

1. Отправьте `/start` боту
2. Если указан канал, подпишитесь на него
3. Нажмите "Играть в Шпиона"
4. Следуйте инструкциям в боте

## 🔧 Дополнительные настройки

- **Rate Limiting**: Ограничение 0.5 секунды между сообщениями (настроено в коде)
- **Таймер игры**: Настраивается через `GAME_TIMER_DURATION` в секундах
